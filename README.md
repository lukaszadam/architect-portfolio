# Atelier - Architecture Portfolio Website

A modern, elegant architecture portfolio website based on a professional Figma design.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern Aesthetic**: Dark theme with sophisticated typography and imagery
- **Smooth Animations**: Parallax effects, fade-in animations, and hover interactions
- **Portfolio Showcase**: Grid layout for displaying architectural projects
- **About Section**: Comprehensive studio information with statistics

## Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive features and animations

## Usage

Simply open `index.html` in your web browser to view the website locally.

## Sections

1. **Navigation** - Fixed header with logo and menu button
2. **Hero Section** - Full-screen introduction with striking tagline
3. **About Section** - Studio information, philosophy, and achievements
4. **Selected Works** - Portfolio grid showcasing 6 featured projects
5. **Footer** - Contact information and social links

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)
- Vanilla JavaScript
- Unsplash images for placeholders

## Customization

You can easily customize:
- Colors in the CSS `:root` variables
- Images by replacing the Unsplash URLs
- Content in the HTML file
- Project information in the works grid

## Browser Support

Works in all modern browsers:
- Chrome
- Firefox
- Safari
- Edge
